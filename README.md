# Landing Martinez & Scienza

Proyecto multipagina listo para deploy con Vite.

## Desarrollo local

```bash
npm install
npm run dev
```

## Build de produccion

```bash
npm run build
```

Salida en `dist/`:
- `dist/index.html`
- `dist/quienes-somos.html`
- `dist/assets/*`

## Preview local de build

```bash
npm run preview
```

## Nota

`landing_martinezscienza.html` queda como archivo de referencia historica de diseno. El sitio activo para deploy usa:
- `index.html`
- `quienes-somos.html`
- `scss/main.scss`
- `js/app.js`
